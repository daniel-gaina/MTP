We have the following categories:
\begin{itemize}
\item
Spec, for a theory.

\item
ME, for module expressions.

\item
Decs, for the declarations in the theory.

\item
TId, for theory identifiers.

\item
PId, for parameter identifiers.

\item
Params, for defining theory parameters.
\end{itemize}
